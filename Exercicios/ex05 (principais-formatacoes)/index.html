Principais formatações

Negrito

Vou deixar a palavra Negrito em Negrito usando uma tag não semantica, ou seja, deve-se evitar essa tag B

Vou deixar a palavra Negrito em Negrito usando uma tag semantica, ou seja, deve-se usar essa tag Strong

Itálico

Vou deixar a palavra Destaque em italico usando uma tag não semantica, ou seja,deve-se evitar usar essa tag I

Vou deixar a palavra Destaque em italico usando uma tag semantica, ou seja, deve-se usar essa tag EM

Texto marcado

Vou deixar a palavra Marca Texto marcado nesse texto estilo marca texto tradicional com a tag MARK

Texto garnde e pequeno

Vou deixar a palavra Grande marcado como palavra grande com a tag BIG

Vou deixar a palavra Pequena marcada como uma palavra pequena com a tag SMALL

Texto deletado - Riscado no meio

Vou deixar a palavra Riscada como uma palavra deletada, ou seja com um risco no meio feita com a tag DEL

Texto inserido - risco a baixo da palavra

Vou deiar a palavra Inserida como uma palavra inserida com a tag INS

Texto Sobrescrito - Palavra a cima

Vou deixa o número X 20 elevado a cima do X usando a tag SUP

Texto subescrito - palavra a baixo

Vou deixar o número H 2 sob a letra H usando a tag SUB

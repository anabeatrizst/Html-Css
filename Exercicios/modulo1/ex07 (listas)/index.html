Listas

Listas Ordenadas

Uma lista ordenada é uma lista onde os itens tem sequencia númerica ou alfabetica. Segue o exemplo:

  1. acordar
  2. escovar os dentes
  3. pentear o cabelo
  4. trocar de roupa
  5. pegar chaves
  6. ir trabalhar

Listas não Ordenadas

Uma lista não ordenada é uma lista onde não precisar ter uma sequencia. Segue o exemplo:

  • Acordar
  • Escovar os dentes
  • Pentear o cabelo
  • Trocar de roupa
  • Pegar chaves
  • Ir trabalhar
